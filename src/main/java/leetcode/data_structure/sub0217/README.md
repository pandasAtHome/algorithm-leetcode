# 题目

- Java：[217]存在重复元素
- Related Topics 数组 哈希表 排序 👍 918 👎 0

```text
给你一个整数数组 nums 。
  如果任一值在数组中出现 至少两次 ，返回 true ；
  如果数组中每个元素互不相同，返回 false 。


示例 1： 

输入：nums = [1,2,3,1]
输出：true 


示例 2： 

输入：nums = [1,2,3,4]
输出：false 


示例 3： 

输入：nums = [1,1,1,3,3,4,3,2,4,2]
输出：true 


提示： 

1 <= nums.length <= 10⁵ 
-10⁹ <= nums[i] <= 10⁹ 
```

# 读题

- 判断一个数组中，是否存在重复的数字

# 法一：Hash 表

- 核心：Hash 表

## 思路

- 创建一个 hash表
- 遍历所有元素，若 hash 表中已存在，则返回 true
