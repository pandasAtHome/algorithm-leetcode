```text
给定一个未排序的整数数组 nums ，
  找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。 
请你设计并实现时间复杂度为 O(n) 的算法解决此问题。 


示例 1： 

输入：nums = [100,4,200,1,3,2]
输出：4
解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。 


示例 2： 

输入：nums = [0,3,7,2,5,8,4,6,0,1]
输出：9


提示： 

0 <= nums.length <= 10⁵ 
-10⁹ <= nums[i] <= 10⁹ 
```

# 读题

- 在一个乱序的整数数组中，计算数字连续的最长序列的长度

# 法一：双端队列

- 核心：双端队列

## 思路

- 把数组中的元素写入 `hash` 表
- 遍历所有数字
  - 找出该数字所属连续序列的左右边界
  - 并把这些值从 `hash` 表中移除
- 计算连续序列的长度，确定最长序列长度

## 实现

### 边界问题

- `nums.length <= 1`：最大长度 = `nums.length`

### 细节问题

- 移除检测过的边界值
