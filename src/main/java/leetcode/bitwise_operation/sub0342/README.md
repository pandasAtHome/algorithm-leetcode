# 读题

- 判断一个整数是否为 4 的幂次方

# 法一：位运算

- 核心：位运算（清除奇数位）

## 思路

- 判断 n 是否为正数
- 是否为 2 的幂次方（n 为 4的幂次方 等价于 n 为 2的幂次方）
- 二进制表示中，只有 一个 1，且只出现在奇数位（即：`n & 1010 1010 == 0`）
