# 读题

- 对一个数字的二进制码进行反转

# 法一：位运算

- 核心：位运算（位移动）

## 思路

- 向左移动，空出新的位置，用于保存新的位
- 保存 n 的最后一位二进制值（0 or 1）
- n 右移，移除已保存的一位二进制值
